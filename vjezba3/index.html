Osnove HTML i CSS

Osnove HTML

  • HTML - Hypertext Markup Language
  • Opisni jezik koji browser-u opisuje što prikazati i na koji način
  • Verzije: HTML 4.01 - XHTML - HTML5
  • Novo u HTML5 je: Novi tag-ovi, novi atributi
  • Osnovni strukturni tag-ovi: html, head, body
  • Osnovni sadržajni tag-ovi: h1, p, ul, strong, li, span, table
  • Postoje jednostruki tag-ovi: img, hr, br
  • Oblikovanje HTML-a: radimo pomoću stilskog jezika CSS

Osnove CSS-a

  • CSS - Cascading Style Sheet
  • Stilski jezik kojim oblikujemo HTML
  • Verzije: CSS1, CSS2, CSS3
  • Novo u CSS3 su nova svojstva
  • Vanjski CSS, unutarnji CSS i inline CSS
  • Unutarnji CSS je unutar HTML dokumenta i style tag-a u head-u
  • CSS pravilo se sastoji od selektora, svojstva i vrijednosti
  • Tri tipa selektora: Tag selektor, ID selektor, Class selektor
