Osnove PHP-a

  • PHP (Hypertext preprocessor) je programski jezik raširen i popularan pri izradi web stranica
  • Originalno je skraćenica PHP značila Personal Home Page Tools, a kasnije je značenje promijenjeno i danas znači PHP: Hypertext Preprocessor (PHP: hipertekstualni pretprocesor, što opisuje glavnu funkciju jezika PHP - da na temelju PHP naredbi generira HTML, jezik kojim se opisuje hipertekst.)
  • PHP naredbe najčešće se nalaze umetnute unutar HTML kôda zajedno s HTML oznakama
